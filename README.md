# Renile Task
## Installation

1. Install dependencies: `npm install`
2. Start application: `npm start `
3. https://explore.postman.com/templates/10327/renilecollection go to this url to get api and use it in postman

### Docker

1. Change the ip address of mongodb from 127.0.0.1:27017  to your ip address of mongodb in docker
2. Run: `docker-compose up --build`


### Test Data
 run `npm run test`.
